# Prog3D_project
Delattre Thibaut / Defoort Nicolas / Mege Mathias
Le jeu consiste à trouver 3 clés sur la map afin d'accéder à un portail qui donne accès à la salle aux trésors.
Une clé est cachée, l'autre générée en tuant tous les ennemis et la dernière en détruisant toutes les cibles qui apparaitront une fois les ennemis abattus.
Des murs peuvent être trompeurs ;)
